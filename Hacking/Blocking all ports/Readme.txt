This exe file can be used to block and unblock all unused ports in windows 7, 8, 8.1, 10
It is a simple command-line software
Press Y to Block all unused ports
Press N to Unblock all unused ports
Press O to exit the command-line prompt

In this zip file contains four files
1. usb.exe
2.run.bat
3.readme.txt
4. Code in python

If the usb.exe is not running directly, then run it through run.bat

Requirments: 
Microsoft Visual C++ Redistributable
